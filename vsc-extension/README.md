# Jano Fidel (ጃኖ ፊደል) - VS Code Support

This extension provides official language support for **Jano Fidel**, a native Amharic programming language built on Node.js.

## Features

- **Syntax Highlighting**: Beautiful color support for Amharic keywords like `ይሁን`, `ተግባር`, and `ከሆነ`.
- **Snippet Support**: Quickly scaffold Amharic code blocks.
- **Language Integration**: Automatically detects `.jf` files.

## Requirements

You must have the Jano Fidel compiler installed globally to run code:
```bash
npm install -g janofidel
```
## How to use
1. Install this Jano fidel extension
2. Create a file ending in ```.jf ``` (e.g, ```ዋና.jf```).
3. Start coding in amharic.

## Extension Settings

This extension contributes to the following settings:

```jano.enable ``` : Enable/disable this extension.
